# WinFormsApp1 – Quản lý KTX

## Mô tả
Ứng dụng WinForms quản lý ký túc xá:
- Sinh viên
- Phòng
- Hợp đồng
- Hóa đơn
- Phiếu báo hỏng

## Công nghệ
- C#
- .NET 8
- WinForms
- Mô hình 3 lớp (GUI – BLL – DAL)

## Cách chạy
1. Mở file `WinFormsApp1.sln` hoặc `WinFormsApp1.csproj`
2. Build & Run bằng Visual Studio
